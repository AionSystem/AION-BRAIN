# Use Case 006: Workplace Grief Response Protocol

**Engine:** Crisis & Grief Engine v1.5
**Domain:** Organizational Response
**Complexity:** Medium
**Risk Level:** MEDIUM (HR/organizational context)

---

## Scenario Description

An HR professional needs guidance on supporting employees experiencing grief, managing bereavement leave, and responding to workplace deaths. The engine provides framework guidance, not legal or policy advice.

---

## Input/Output Example

### Engine Response
```
WORKPLACE GRIEF RESPONSE FRAMEWORK
==================================

[EDUCATIONAL FRAMEWORK - HR/Legal Review Required]

SECTION 1: INDIVIDUAL EMPLOYEE GRIEF SUPPORT
────────────────────────────────────────────

When an Employee Experiences a Loss:

IMMEDIATE RESPONSE (First 24-48 hours):
├─ Express genuine condolences
│   └─ "I'm so sorry for your loss"
├─ Offer practical support
│   └─ "What do you need right now?"
├─ Explain available leave
│   └─ Bereavement policy, FMLA if applicable
├─ Handle workload
│   └─ Temporary coverage, deadline extensions
├─ Provide EAP information
│   └─ If organization has Employee Assistance Program
└─ Respect privacy
    └─ Ask before sharing with team

RETURN TO WORK SUPPORT:
├─ Flexible re-entry options if possible
├─ Reduced expectations initially
├─ Regular check-ins (without intrusion)
├─ Watch for signs of complicated grief
├─ Ongoing EAP availability
└─ [VERIFY_REQUIRED:hr_policy_compliance]

MANAGER GUIDANCE:
├─ DO:
│   ├─ Acknowledge the loss directly
│   ├─ Offer specific help
│   ├─ Be patient with performance
│   ├─ Follow up over time (grief is long)
│   └─ Know your resources (HR, EAP)
│
└─ DON'T:
    ├─ Avoid mentioning the loss
    ├─ Rush return to normal
    ├─ Compare losses
    ├─ Offer unsolicited advice
    └─ Expect timeline for "getting over it"

SECTION 2: WORKPLACE DEATH RESPONSE
───────────────────────────────────

When a Coworker Dies:

IMMEDIATE ACTIONS (Day 1):
├─ Verify information before communicating
├─ Notify leadership/HR immediately
├─ Prepare communication plan
├─ Arrange coverage for critical functions
├─ Contact family (if appropriate relationship exists)
└─ [VERIFY_REQUIRED:organizational_protocol]

COMMUNICATION PRINCIPLES:
├─ FIRST: In-person or phone to close colleagues
├─ THEN: Department/team notification
├─ THEN: Broader organization
├─ TONE: Compassionate, factual, not overly detailed
├─ RESPECT: Family wishes regarding cause of death
└─ RESOURCES: Include support resources in communication

Sample Communication Framework:
├─ Announce the loss with appropriate solemnity
├─ Share basic facts (as appropriate/authorized)
├─ Acknowledge impact on team
├─ Provide immediate support resources
├─ Share memorial/funeral information when available
└─ [VERIFY_REQUIRED:legal_review_of_communication]

SUPPORT RESPONSE:
├─ EAP Critical Incident Response
├─ Grief counselors available on-site (if traumatic)
├─ Private space for employees who need it
├─ Permission to leave if overwhelmed
├─ Team meeting for processing (optional attendance)
└─ Follow-up support over time

PRACTICAL MATTERS:
├─ Workstation/personal effects handling
├─ Email/account management
├─ Ongoing projects/responsibilities
├─ Memorial or acknowledgment within organization
└─ [VERIFY_REQUIRED:policy_and_legal_review]

SECTION 3: DEATH BY SUICIDE CONSIDERATIONS
──────────────────────────────────────────

Additional Protocols for Suicide:

COMMUNICATION:
├─ Follow family wishes on disclosure
├─ If disclosed: "died by suicide" (not "committed")
├─ Avoid details of method or location
├─ Do not sensationalize or romanticize
├─ Include crisis resources in all communications
└─ [VERIFY_REQUIRED:suicide_postvention_expert]

ENHANCED SUPPORT:
├─ Crisis counselors on-site essential
├─ Screen for at-risk individuals
│   └─ Close colleagues, those with similar struggles
├─ Provide Survivors of Suicide Loss resources
├─ Watch for contagion risk
├─ Longer-term support often needed
└─ [MANDATORY: Consult mental health professional]

CONTAGION PREVENTION:
├─ Don't glorify or oversimplify
├─ Focus on prevention resources
├─ Acknowledge complexity
├─ Professional consultation required
└─ [CRITICAL: Not DIY—get expert help]

SECTION 4: BEREAVEMENT LEAVE CONSIDERATIONS
───────────────────────────────────────────

Policy Framework Elements:
├─ COVERED RELATIONSHIPS
│   ├─ Immediate family (typically 3-5 days)
│   ├─ Extended family (typically 1-3 days)
│   ├─ Chosen family considerations
│   └─ Pregnancy loss inclusion
│
├─ FLEXIBILITY OPTIONS
│   ├─ Consecutive vs. intermittent leave
│   ├─ Extension for travel
│   ├─ Cultural/religious observance accommodation
│   └─ Additional unpaid leave option
│
├─ BEYOND BEREAVEMENT LEAVE
│   ├─ FMLA for serious health condition
│   ├─ Short-term disability (if grief-related condition)
│   ├─ ADA accommodation (if grief impacts functioning)
│   └─ Flexible work arrangements
│
└─ DOCUMENTATION
    ├─ Balance verification needs with compassion
    ├─ Obituary or funeral program typically sufficient
    ├─ Don't create administrative burden during grief
    └─ [VERIFY_REQUIRED:hr_policy_compliance]

═══════════════════════════════════════════════
⚠️ ORGANIZATIONAL DISCLAIMER
═══════════════════════════════════════════════

This framework requires:
├─ HR policy review and alignment
├─ Legal counsel review (employment law)
├─ EAP provider coordination
├─ Leadership approval for protocols
└─ Cultural adaptation for organization

Not legal or employment advice.

[VERIFY_REQUIRED:hr_legal_policy_review]
```
