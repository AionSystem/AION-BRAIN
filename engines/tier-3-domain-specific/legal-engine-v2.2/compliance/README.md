## Documentation Requirements

### Minimum Retention

| Document Type | Retention Period |
|---------------|-----------------|
| AI-assisted work product | Per jurisdictional rules (typically 5-7 years) |
| Verification records | Same as work product |
| Client consent | Same as client file |
| Audit trails | Per firm policy (recommended: 7 years) |

### Recommended Documentation

| Element | Purpose |
|---------|---------|
| AI tool identification | Which tools used |
| Supervision record | Who reviewed output |
| Verification completion | What was verified |
| Client notification | Consent documented |
