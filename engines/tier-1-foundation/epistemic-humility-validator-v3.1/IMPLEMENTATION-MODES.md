## Practical LLM Implementation Tips

### For Completeness Detection:
```
Search for absolute language:
"Scan for: all, every, complete, exhaustive, comprehensive, 
definitely, certainly, proven, always, never"
Flag each instance for Gödel constraint check
```

### For Prediction Horizon:
```
Request explicit time bounds:
"What is the system type: Simple/Complex/Chaotic/Adaptive?
Based on type, what is the maximum reliable prediction horizon?
Show confidence decay at Year 1, 3, 5, 10."
```

### For Bayesian Quantification:
```
Convert vague to specific:
"Replace 'probably' with 'X-Y% (95% CI)'
Replace 'likely' with 'P > 0.6 based on [reasoning]'
Replace 'might' with 'P ~ 0.3-0.5, uncertain'"
```

### For Cross-Domain Validation:
```
Structure the analysis:
"1. What domain does this reasoning come from?
2. What domain is it being applied to?
3. Do the mechanisms match?
4. Do the boundary conditions match?
5. What could go wrong if transfer is invalid?"
```
