## Contribution Guidelines

### Submitting Use Cases

Use cases from real practice (properly de-identified) are valuable contributions.

**Requirements:**
1. Complete de-identification (HIPAA compliant)
2. Professional context clearly stated
3. Safety elements highlighted
4. Cultural sensitivity addressed
5. Learning value articulated

**Submission:** AIONSYSTEM@outlook.com with subject "Use Case Submission"

---

## Important Notes

1. **All scenarios are educational** — Not prescriptive protocols
2. **Professional judgment required** — Use cases illustrate, not dictate
3. **Context matters** — Adapt to specific situation
4. **Safety first** — Never compromise on safety elements
5. **Cultural humility** — Recognize diversity within examples

---

*Use cases are learning tools. Real crisis intervention requires training, supervision, and ongoing professional development.*
