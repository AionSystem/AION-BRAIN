# Malpractice Shield Module (MSM-v1.0)

**Classification:** Legal Engine v2.2 Enhancement Module
**Purpose:** Proactive malpractice risk detection and documentation for attorney protection

---

## 1. Module Overview

The Malpractice Shield Module identifies, documents, and mitigates malpractice risk factors in real-time, creating defensible records of attorney diligence when using AI tools in legal practice.

### Core Mission

| Function | Description |
|----------|-------------|
| Risk Detection | Identify malpractice exposure patterns |
| Documentation | Create contemporaneous defense records |
| Mitigation Guidance | Suggest risk-reduction measures |
| Insurance Alignment | Support carrier compliance requirements |

### 2.1 Top Malpractice Triggers

| Category | Frequency | AI-Related Risk |
|----------|-----------|-----------------|
| Missed Deadlines | 24% of claims | Calendar/reminder failures |
| Inadequate Investigation | 18% | Over-reliance on AI research |
| Failure to Know Law | 15% | AI hallucination/outdated info |

| Conflicts of Interest | 9% | Inadequate conflict checking |

| Other | 7% | Various |

### 2.2 AI-Specific Risk Amplifiers

| Risk | Description | Mitigation |
|------|-------------|------------|
| Citation Fabrication | AI invents non-existent cases | Shepardize/KeyCite verification |
| Hallucinated Holdings | AI misrepresents case holdings | Read cited cases in full |
| Stale Authority | AI uses overruled precedent | Verify current status |
| Jurisdiction Confusion | AI applies wrong state law | Explicit jurisdiction checking |
| Confidentiality Breach | PII in prompts | Redaction protocols |
| Over-Delegation | AI replaces judgment | Maintain attorney control |

## 3. Real-Time Risk Scoring

### 3.1 Risk Level Classification

| Level | Indicators | Required Action |
|-------|------------|-----------------|
| **CRITICAL** | Fabricated citation used, deadline imminent, sanctions risk | Immediate attorney review, halt output |
| **HIGH** | Uncited legal proposition, potential ethics violation | Enhanced verification, supervisor review |
| **MODERATE** | Standard AI output, routine matter | Normal verification protocol |
| **LOW** | Educational use, internal memo only | Standard documentation |
