## How to Submit

### Step 1: Prepare Your Submission

Use this template:

```markdown
## Use Case: [Title]

### Summary
[2-3 sentence overview]

### Problem
[What problem were you solving?]

### Solution
[How did you use AION-BRAIN?]

### Engines Used
[List the engines you used]

### Results
[What outcomes did you achieve?]

### Lessons Learned
[What would you do differently?]

### About the Submitter
[Name/Role - optional, can be anonymous]
```

### Step 2: Submit

**Option A: GitHub Discussions**
1. Go to Discussions → Use Cases
2. Create new discussion with your use case
3. Tag with relevant labels

**Option B: Email**
1. Email: AIONSYSTEM@outlook.com
2. Subject: "Use Case Submission: [Title]"
3. Include template content

### Step 3: Review

1. We review submissions for completeness
2. We may ask clarifying questions
3. Approved submissions are added here
4. You're notified when published

---

## Submission Guidelines

### Required

- Real implementation (not hypothetical)
- Clear problem and solution
- At least one engine used
- Some form of results/outcome

### Encouraged

- Quantitative results
- Detailed implementation notes
- Lessons learned
- Before/after comparison

### Privacy Options

| Option | What's Published |
|--------|------------------|
| **Full attribution** | Your name, organization, details |
| **Partial** | Your name only, no organization |
| **Anonymous** | Use case only, no personal info |
| **Private** | Not published, internal reference |
