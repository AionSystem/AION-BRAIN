## How to Participate

### Via GitHub Discussions

1. Navigate to the repository's **Discussions** tab
2. Choose the appropriate category
3. Start a new discussion or join existing ones
4. Follow the community guidelines

### Via These Pages

These markdown files serve as:
- **Templates** for discussion topics
- **Guidelines** for each category
- **Archives** of notable discussions
- **FAQ** for common questions

## Community Guidelines

### Do

| Action | Description |
|--------|-------------|
| **Be respectful** | Treat all community members with courtesy |
| **Be constructive** | Offer actionable feedback and suggestions |
| **Be specific** | Provide details, examples, and context |
| **Search first** | Check if your topic has been discussed |
| **Stay on topic** | Use appropriate categories |

### Don't

| Action | Description |
|--------|-------------|
| **Personal attacks** | Criticize ideas, not people |
| **Spam** | No promotional content without value |
| **Duplicate topics** | Consolidate related discussions |
| **Off-topic content** | Keep discussions relevant |
| **Confidential info** | Don't share sensitive data |

### Moderation

- Maintainers may move discussions to appropriate categories
- Violations of guidelines may result in warnings or removal
- Appeals can be sent to: AIONSYSTEM@outlook.com

## Discussion Templates

### Feature Request Template

```markdown
## Feature Request: [Title]

### Problem
What problem does this feature solve?

### Proposed Solution
How would you like this solved?

### Alternatives Considered
What other solutions have you considered?

### Additional Context
Any other relevant information?
```

### Bug Report Template

```markdown
## Bug Report: [Title]

### Engine/Component
Which engine or component is affected?

### Steps to Reproduce
1. Step one
2. Step two
3. ...

### Expected Behavior
What should happen?

### Actual Behavior
What actually happens?

### Environment
- Model used:
- Platform:
- Version:
```
