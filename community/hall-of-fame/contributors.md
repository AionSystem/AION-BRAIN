## How to Become a Contributor

### Contribution Types

| Type | Description | Recognition |
|------|-------------|-------------|
| **Documentation** | Improve docs, fix typos, add examples | Listed in Documentation Contributors |
| **Code** | Improve engines, add features | Listed in Engine Contributors |
| **Translation** | Translate to other languages | Listed in Translation Contributors |
| **Integration** | Build integrations with other tools | Listed in Integration Contributors |
| **Community** | Help others, answer questions | Listed in Community Helpers |
| **Research** | Academic papers, citations | Listed in Research Collaborators |

### Contribution Process

1. **Find an opportunity** — Check Issues or Discussions
2. **Fork the repository** — Create your own copy
3. **Make your changes** — Follow our guidelines
4. **Submit a Pull Request** — Describe your contribution
5. **Review process** — Maintainers review and provide feedback
6. **Merge and recognition** — You're now a contributor!

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for detailed guidelines.
