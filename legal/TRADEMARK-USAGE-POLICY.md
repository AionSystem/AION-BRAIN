### 1.4 Trademark vs. Copyright

| Aspect | Trademark | Copyright (Apache 2.0) |

| What it protects | Brand identity, logos, names | Software code, documentation |
| Open-source inclusion | NOT included | Included |
| Permission required | Yes, for certain uses | Granted by license |
| Purpose | Prevents brand confusion | Enables software sharing |

## 2. Permitted Uses (No Permission Required)

### 2.1 Nominative Fair Use

You MAY use AION-BRAIN trademarks without permission for:

**Truthful Reference:**
- "This application uses AION-BRAIN cognitive infrastructure"
- "Built with AION-BRAIN open-source engines"
- "Compatible with AION-BRAIN frameworks"

**Commentary and Criticism:**
- Reviews, comparisons, and analysis
- Academic research and publications
- News reporting and journalism

**Community Activities:**
- User group names (e.g., "AION-BRAIN Users of Chicago")
- Educational presentations about AION-BRAIN
- Open-source contribution discussions

### 2.2 Requirements for Permitted Use

Even for permitted uses, you must:

| Requirement | Description |
|-------------|-------------|
| Accuracy | Only make truthful statements about AION-BRAIN |
| Attribution | Acknowledge AION-BRAIN ownership when appropriate |
| No Endorsement Implication | Don't suggest AION-BRAIN endorses you |
| Trademark Notice | Include ™ symbol on first prominent use |
| Distinguish | Make clear what is yours vs. what is AION-BRAIN |

### 2.3 Examples of Permitted Use

**Acceptable:**
- "MyApp integrates with AION-BRAIN™ cognitive engines"
- "An analysis of AION-BRAIN's approach to AI safety"
- "AION-BRAIN User Meetup - Boston Chapter"
- "This research extends the AION-BRAIN Medical Engine methodology"

**NOT Acceptable (see Prohibited Uses):**
- "AION-BRAIN MyApp" (implies official product)
- "Official AION-BRAIN Partner" (without authorization)
- Using AION-BRAIN logo as your app icon

## 4. Prohibited Uses

### 4.1 Never Permitted

The following uses are NEVER permitted, regardless of partnership status:

| Prohibited Use | Reason |
|----------------|--------|
| Using marks in your product name | Creates confusion (e.g., "AION-BRAIN Pro") |
| Modifying our logos | Dilutes brand identity |
| Using marks in domain names | Cybersquatting concerns |
| Suggesting official endorsement falsely | Deceptive to consumers |
| Using marks for illegal purposes | Harms brand reputation |
| Using marks for competing products | Unfair competition |
| Registering our marks as your trademarks | Trademark violation |

### 4.2 Product Naming Restrictions

**DO NOT name your product:**
- AION-BRAIN [Anything]
- [Anything] AION-BRAIN
- AI/ON [Anything]
- AION [Anything] (in AI/cognitive context)

**Acceptable alternatives:**
- [YourBrand] for AION-BRAIN
- [YourBrand] powered by AION-BRAIN
- [YourBrand] (compatible with AION-BRAIN)

### 4.3 Logo Usage Restrictions

| Restriction | Details |
|-------------|---------|
| No modifications | Don't alter colors, proportions, or elements |
| No combinations | Don't merge with other logos |
| No backgrounds | Don't place on busy or conflicting backgrounds |
| Minimum size | Maintain legibility (minimum 24px height for digital) |
| Clear space | Maintain minimum clear space around logo |
