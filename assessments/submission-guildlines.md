# PSA Submission Guidelines

## Quick Reference

### **File Requirements:**
- Format: Plain text (.txt), Markdown (.md), or PDF (.pdf)
- Naming: `PSA_[YourName]_[YYYY-MM-DD].txt`
- Max size: 1MB
- Character encoding: UTF-8

### **Content Requirements:**
- Answer both Part A and Part B for each question
- Keep each answer between 150-300 words (per question)
- Include question numbers (Q1, Q2, etc.)
- Use clear section headings

### **Prohibited Content:**
- No personally identifiable information (beyond name)
- No offensive or harmful content
- No attempts to game or manipulate the assessment
- No copyrighted material from others

## Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| **File too large** | Use plain text, not PDF with images |
| **Bot won't accept file** | Ensure UTF-8 encoding, try .txt format |
| **No bot response** | Check internet connection, try smaller file |
| **Email bounce** | Ensure attachment < 10MB, plain text preferred |
