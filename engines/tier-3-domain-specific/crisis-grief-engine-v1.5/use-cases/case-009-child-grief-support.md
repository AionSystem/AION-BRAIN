# Use Case 009: Children's Grief Support Framework

**Engine:** Crisis & Grief Engine v1.5
**Domain:** Developmental Grief
**Complexity:** High
**Risk Level:** HIGH (child welfare)

---

## Scenario Description

A school counselor needs educational content about age-appropriate grief support for children. Professional guidance context only, not direct child interaction.

CHILDREN'S GRIEF SUPPORT FRAMEWORK
==================================

[PROFESSIONAL EDUCATIONAL RESOURCE]

SECTION 1: DEVELOPMENTAL UNDERSTANDING
──────────────────────────────────────

Children's Grief by Developmental Stage:

INFANTS AND TODDLERS (0-2 years):
├─ Conceptual Understanding: None
├─ Grief Expression:
│   ├─ React to caregiver distress
│   ├─ Separation anxiety
│   ├─ Regression in development
│   ├─ Changes in eating/sleeping
│   └─ Irritability, clinginess
├─ Support Approach:
│   ├─ Maintain routines
│   ├─ Consistent caregiving
│   ├─ Physical comfort
│   └─ Calm environment
└─ Note: They grieve absence, not death concept

PRESCHOOL (3-5 years):
├─ Conceptual Understanding:
│   ├─ Magical thinking
│   ├─ Death not permanent
│   ├─ May believe they caused it
│   └─ Concrete, literal understanding
├─ Grief Expression:
│   ├─ Repeated questions
│   ├─ Play themes involving death
│   ├─ Regression (bedwetting, thumb-sucking)
│   ├─ Sleep disturbance, nightmares
│   └─ Brief grief bursts
├─ Support Approach:
│   ├─ Simple, concrete explanations
│   ├─ Avoid euphemisms ("went to sleep")
│   ├─ Answer questions honestly
│   ├─ Reassurance of care and safety
│   └─ Play-based expression
└─ Note: Will need repeated explanations

SCHOOL AGE (6-11 years):
├─ Conceptual Understanding:
│   ├─ Death is permanent
│   ├─ Death has causes
│   ├─ Can happen to anyone
│   ├─ Worry about own death or others
│   └─ Concrete operations
├─ Grief Expression:
│   ├─ Questions about death details
│   ├─ School performance changes
│   ├─ Physical complaints
│   ├─ Acting out or withdrawal
│   └─ May hide feelings to protect adults
├─ Support Approach:
│   ├─ Honest, age-appropriate information
│   ├─ Permission to ask questions
│   ├─ Encourage expression (art, writing)
│   ├─ Maintain routines and activities
│   └─ Grief groups with peers
└─ Note: May need permission to grieve

ADOLESCENTS (12-18 years):
├─ Conceptual Understanding:
│   ├─ Adult understanding of death
│   ├─ Abstract thinking about meaning
│   ├─ Future impact awareness
│   └─ May challenge religious/spiritual beliefs
├─ Grief Expression:
│   ├─ Mood swings, irritability
│   ├─ Risk-taking behavior
│   ├─ Isolation or peer focus
│   ├─ Academic changes
│   └─ Identity exploration
├─ Support Approach:
│   ├─ Respect autonomy while staying connected
│   ├─ Peer support options
│   ├─ Multiple modes of expression
│   ├─ Monitor for depression/risk
│   └─ Include in family processes
└─ Note: May grieve in privacy; still need support

SECTION 2: TALKING TO CHILDREN ABOUT DEATH
──────────────────────────────────────────

Guidance for Adults:

LANGUAGE PRINCIPLES:
├─ Use clear, simple language
├─ Use the word "died" (not euphemisms)
├─ Age-appropriate honesty
├─ Answer questions directly
├─ "I don't know" is acceptable
└─ [TRAINING_REQUIRED:developmentally_appropriate_language]

AVOID:
├─ "Went to sleep" → Fear of sleeping
├─ "God took them" → Anger at God or fear
├─ "They went away" → Fear of separation
├─ "They were so good" → Fear of being good
├─ "It's for the best" → Minimizes loss
└─ Overly detailed explanations

SAMPLE EXPLANATIONS (Guidance Only):
├─ "Dad's body stopped working and can't be fixed"
├─ "Grandma's heart was very sick and stopped"
├─ "The accident hurt her body too much to heal"
├─ "He had a sickness called cancer"
└─ [VERIFY_REQUIRED:tailor_to_child_and_situation]

RESPONDING TO QUESTIONS:
├─ "Will you die too?" 
│   └─ "I plan to be here for a very long time"
├─ "Did I cause it?"
│   └─ "No, nothing you did caused this"
├─ "Where are they now?"
│   └─ [Family/cultural beliefs appropriate]
├─ "Can they come back?"
│   └─ "No, when someone dies, they can't come back"
└─ "Why did they die?"
    └─ [Age-appropriate, honest answer]

SECTION 3: SCHOOL SUPPORT FRAMEWORK
───────────────────────────────────

School Response Protocol:

WHEN A STUDENT LOSES A LOVED ONE:
├─ COORDINATE with family
│   ├─ Confirm facts
│   ├─ Ask permission to share with class
│   ├─ Understand family wishes
│   └─ Get return plan
│
├─ INFORM relevant staff
│   ├─ Teachers, counselor, admin
│   ├─ Share appropriate information
│   ├─ Plan for support
│   └─ Monitor consistently
│
├─ SUPPORT classmates
│   ├─ Age-appropriate acknowledgment
│   ├─ Cards or project if appropriate
│   ├─ Don't force participation
│   └─ Watch for affected friends
│
├─ RETURN to school support
│   ├─ Welcome back privately
│   ├─ Establish check-in plan
│   ├─ Identify safe person/place
│   ├─ Flexible expectations initially
│   └─ Normalize grief reactions
│
└─ ONGOING support
    ├─ Regular check-ins
    ├─ Anniversary awareness
    ├─ Mother's/Father's Day sensitivity
    ├─ Monitor academic/behavioral changes
    └─ Family communication

WHEN A STUDENT DIES:
├─ Crisis response team activation
├─ Staff notification and support
├─ Age-appropriate classroom discussions
├─ Counselors available
├─ Parent communication
├─ Media management
├─ Memorial planning (with family)
└─ [VERIFY_REQUIRED:crisis_protocol]

├─ Persistent depressive symptoms (>2 weeks)
├─ Suicidal ideation or self-harm
├─ Severe regression lasting more than a few weeks
├─ Complete withdrawal from peers/activities
├─ Significant academic decline
├─ Severe behavioral problems
├─ Trauma symptoms (re-experiencing, avoidance)
├─ Physical symptoms without medical cause
├─ Substance use
└─ [MANDATORY: Mental health professional referral]
