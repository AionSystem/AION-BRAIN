# Use Case 005: Bereavement Support Group Facilitation Guide

**Engine:** Crisis & Grief Engine v1.5
**Domain:** Group Facilitation
**Complexity:** Medium
**Risk Level:** MEDIUM (facilitation guidance)

---

## Scenario Description

A grief support group facilitator needs educational content about group dynamics, session structure, and handling difficult situations. The engine provides facilitation frameworks for trained facilitators, not replacement for professional training.

## Input/Output Example

### Engine Response
```
BEREAVEMENT GROUP FACILITATION GUIDE
====================================

[EDUCATIONAL FRAMEWORK - Training Required]

SECTION 1: GROUP STRUCTURE
──────────────────────────

Typical Session Format (90-120 minutes):
├─ OPENING (10-15 min)
│   ├─ Welcome and check-in
│   ├─ Review group guidelines
│   ├─ Moment of acknowledgment
│   └─ Brief mindfulness/grounding if appropriate
│
├─ SHARING TIME (45-60 min)
│   ├─ Structured sharing (topic-based or open)
│   ├─ Time management for equity
│   ├─ Facilitated discussion
│   └─ Active listening modeling
│
├─ EDUCATIONAL COMPONENT (15-20 min) [Optional]
│   ├─ Grief education topic
│   ├─ Coping skill introduction
│   ├─ Resource sharing
│   └─ Guest speaker (occasionally)
│
├─ CLOSING (10-15 min)
│   ├─ Summary of themes
│   ├─ Self-care reminder
│   ├─ Preview of next session
│   └─ Closing ritual if used
│
└─ POST-SESSION
    ├─ Co-facilitator debrief
    ├─ Documentation
    ├─ Follow-up on concerns
    └─ Self-care for facilitators

GROUP GUIDELINES (Sample):
├─ Confidentiality: "What's shared here stays here"
├─ Respect: Listen without judgment
├─ "I" statements: Speak from own experience
├─ No advice-giving: Unless specifically requested
├─ Right to pass: No one required to share
├─ One voice: No side conversations
├─ Support focus: Not therapy
└─ Crisis resources: Know when to use them

SECTION 2: FACILITATION SKILLS
──────────────────────────────

Key Facilitation Behaviors:
├─ MODELING
│   ├─ Active listening demonstration
│   ├─ Appropriate self-disclosure
│   ├─ Healthy grief expression
│   └─ Boundary maintenance
│
├─ HOLDING SPACE
│   ├─ Create safety through presence
│   ├─ Allow silence
│   ├─ Tolerate distress without fixing
│   └─ Maintain group container
│
├─ BALANCING
│   ├─ Quiet members and talkative members
│   ├─ Individual needs and group needs
│   ├─ Structure and flexibility
│   └─ Support and challenge
│
├─ LINKING
│   ├─ Connect member experiences
│   ├─ Highlight common themes
│   ├─ Build group cohesion
│   └─ "It sounds like several of you..."
│
└─ GATEKEEPING
    ├─ Ensure equitable participation
    ├─ Gently redirect monopolizers
    ├─ Invite quiet members
    └─ Protect vulnerable members

SECTION 3: CHALLENGING SITUATIONS
─────────────────────────────────

Situation: MEMBER IN ACUTE DISTRESS
├─ Acknowledge the distress: "I can see this is really hard"
├─ Offer support: "Would it help to take a breath?"
├─ Check capacity: "Would you like to continue or take a break?"
├─ Normalize: "Grief can hit us unexpectedly in groups"
├─ Don't rush: Allow time for composure
└─ Follow up after group if needed
└─ [ESCALATE if distress indicates safety concern]

Situation: MEMBER MONOPOLIZING
├─ Acknowledge contribution: "Thank you for sharing that"
├─ Redirect: "Let's hear from others on this"
├─ Time limit: "We want to make sure everyone has time"
├─ Private conversation if pattern continues
└─ Review in co-facilitator debrief

Situation: CONFLICT BETWEEN MEMBERS
├─ Intervene calmly: "Let's pause here"
├─ Acknowledge both perspectives
├─ Return to group guidelines
├─ Redirect to shared purpose
├─ Process after group if needed
└─ Consult supervisor if significant

Situation: DISCLOSURE OF SUICIDAL IDEATION
├─ [IMMEDIATE SAFETY PROTOCOL]
├─ Thank person for trusting the group
├─ Acknowledge this requires additional support
├─ Do not process in group setting
├─ Arrange private follow-up immediately after group
├─ Connect to crisis resources
└─ [MANDATORY: Clinical supervisor notification]

Situation: COMPLICATED GRIEF PRESENTATION
├─ Recognize signs:
│   ├─ Prolonged intense symptoms (>12 months)
│   ├─ Inability to function
│   ├─ Persistent yearning/preoccupation
│   └─ No progress over time
├─ Support group may not be sufficient
├─ Gentle individual conversation
├─ Suggest individual grief therapy
└─ [VERIFY_REQUIRED:clinical_referral]

SECTION 4: SPECIAL CONSIDERATIONS
─────────────────────────────────

Loss-Specific Group Considerations:
├─ SUICIDE LOSS
│   ├─ Higher risk in group members
│   ├─ Shame and stigma processing
│   ├─ "Why" questions are normal
│   ├─ Safety screening essential
│   └─ Trained facilitators required
│
├─ CHILD LOSS
│   ├─ Intense, prolonged grief expected
│   ├─ Couples may grieve differently
│   ├─ Guilt common
│   ├─ "Meaning-making" often important
│   └─ Long-term support often needed
│
├─ SUDDEN/TRAUMATIC LOSS
│   ├─ Trauma symptoms may be present
│   ├─ PTSD screening may be needed
│   ├─ Avoid detailed trauma narratives in group
│   └─ Trauma-informed facilitation required
│
└─ DISENFRANCHISED GRIEF
    ├─ Loss not openly acknowledged
    │   └─ Examples: Pet loss, miscarriage, ex-spouse
    ├─ Validation especially important
    ├─ Group provides acknowledgment
    └─ Challenge societal minimization

⚠️ FACILITATOR REQUIREMENTS
═══════════════════════════════════════════════

This guide supplements, does not replace:
├─ Formal facilitator training program
├─ Clinical supervision for grief groups
├─ Organizational policies and procedures
├─ Background in grief and loss
└─ Co-facilitation for support

[VERIFY_REQUIRED:facilitator_training_verification]
```
