# How to Cite AION-BRAIN

## Quick Reference

### **Standard Software Citation:**
```bibtex
@software{Salmon_AION-BRAIN_2026,
  author = {Salmon, Sheldon K.},
  title = {{AION-BRAIN: Cognitive Infrastructure for Human-AI Collaboration}},
  year = {2026},
  version = {1.0.0},
  url = {https://github.com/AION-BRAIN},
  license = {Apache-2.0}
}
```

APA Style:

Salmon, S. K. (2026). AION-BRAIN: Cognitive Infrastructure for Human-AI Collaboration (Version 1.0.0) [Computer software]. Retrieved from https://github.com/AION-BRAIN

MLA Style:

Salmon, Sheldon K. AION-BRAIN: Cognitive Infrastructure for Human-AI Collaboration. Version 1.0.0, 2026, https://github.com/AION-BRAIN.

Citing Specific Engines

When referencing a specific engine, cite both the overall project and the specific engine:

```bibtex
@software{Salmon_OracleLayer_2025,
  author = {Salmon, Sheldon K.},
  title = {{Oracle Layer v1.0}},
  year = {2025},
  version = {1.0},
  note = {Part of AION-BRAIN: Cognitive Infrastructure for Human-AI Collaboration},
  url = {https://github.com/AION-BRAIN/engines/tier-1-foundation/oracle-layer-v1.0}
}
```
