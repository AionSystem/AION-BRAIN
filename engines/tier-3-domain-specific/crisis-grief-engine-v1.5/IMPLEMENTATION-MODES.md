# Crisis & Grief Counseling Engine - Implementation Modes

Different operational modes for different professional contexts.

---

## Mode Overview

| Mode | Context | Safety Level | User Type |
|------|---------|--------------|-----------|
| **Crisis Hotline Mode** | Real-time crisis calls | Maximum | Trained crisis counselors |
| **Clinical Session Mode** | Therapy/counseling sessions | High | Licensed clinicians |
| **Hospice Support Mode** | End-of-life care | High | Hospice professionals |
| **Death Services Mode** | Funeral/memorial services | Moderate | Funeral professionals |
| **Supervision Mode** | Training and oversight | Variable | Supervisors/trainees |
| **Self-Care Mode** | Provider wellness | Moderate | All professionals |

---

## Mode 1: Crisis Hotline Mode

**Context:** Real-time crisis intervention calls
**Safety Level:** Maximum
**Authorized Users:** Trained crisis counselors only

### Features Enabled
- Immediate risk assessment prompts
- Real-time safety planning support
- Emergency services integration guidance
- Caller location/resource matching
- De-escalation technique suggestions
- Lethal means counseling support

### Safety Protocols
- Continuous risk monitoring
- Automatic escalation triggers
- Supervisor notification prompts
- Post-call documentation
- Immediate debrief reminders

### Output Format
```
CRISIS ASSESSMENT
-----------------
Risk Level: [IMMINENT/HIGH/MODERATE/LOW]
Key Indicators: [List]
Protective Factors: [List]
Immediate Action: [Required steps]
Safety Plan Elements: [Collaborative development]
Follow-up Required: [Yes/No - Timeline]
Escalation Needed: [Yes/No - Reason]
```

## Mode Selection Guidelines

### Choose Based On

| Factor | Consideration |
|--------|---------------|
| Context | Where is the interaction happening? |
| User Training | What is the professional's qualification? |
| Client/Family Status | What is the current crisis level? |
| Time Constraints | Real-time vs. planned interaction |
| Documentation Needs | What records are required? |

### Mode Transitions

Modes can transition based on:
- Escalating risk (always upgrade safety level)
- Session context change
- Professional role shift
- Client/family needs evolution

---

## Safety Across All Modes

Regardless of mode, the engine ALWAYS:

1. Monitors for imminent risk indicators
2. Enforces professional boundaries
3. Prompts for human oversight
4. Maintains audit documentation
5. Respects cultural considerations
6. Preserves human dignity
