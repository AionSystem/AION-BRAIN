## Practical LLM Implementation Tips

### For 5-Framework Integration:
```
Structure prompts to explicitly invoke each framework:
"First analyze using Sun Tzu's terrain concepts,
then identify Meadows leverage points,
then map Barabási network dynamics..."
```

### For Moat Durability Index:
```
Request explicit scoring:
"Score each of these 6 dimensions from 0-10 with reasoning:
1. Imitation Difficulty
2. Switching Costs
3. Network Effects
4. Scale Advantages
5. Brand/Trust
6. Regulatory Protection"
```

### For Competitive Response:
```
Provide competitor context:
"Given that [Competitor A] has [X resources] and [Y pattern],
what is the probability distribution of their responses to [Your Move]?"
```

### For Causal Assumptions:
```
Force assumption surfacing:
"What are the 3 things that MUST be true for this strategy to work?
For each, rate the evidence strength as Strong/Moderate/Weak."
```
