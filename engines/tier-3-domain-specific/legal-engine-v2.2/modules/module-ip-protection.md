## 6. IP Litigation Strategy

### 6.1 Forum Selection for IP Cases

| Forum | Advantages | Considerations |
|-------|------------|----------------|
| E.D. Texas | Patent-friendly, fast | Venue challenges post-TC Heartland |
| D. Delaware | Experienced judges, corporate defendants | Slower docket |
| NDCA | Tech-savvy, IP specialists | Can be defendant-friendly |
| ITC | Fast, import exclusion | No damages, different standards |
| PTAB | Cheaper, claim-focused | Limited estoppel issues |

### 6.2 Remedies Analysis

| Remedy | Availability | Key Factors |
|--------|--------------|-------------|
| Injunction | eBay factors | Irreparable harm, inadequate remedy at law |
| Damages (Lost Profits) | Direct competition | But-for causation, Panduit factors |
| Damages (Reasonable Royalty) | Always available | Georgia-Pacific factors |
| Enhanced Damages | Willfulness | Halo/Seagate standards |
| Attorney Fees | Exceptional case | Octane Fitness standard |

## 7. IP Transaction Support

### 7.1 License Structure

| License Type | Characteristics | Typical Terms |
|--------------|-----------------|---------------|
| Exclusive | Sole licensee, even as to licensor | Higher royalty, performance obligations |
| Sole | One licensee, licensor retains rights | Moderate terms |
| Non-exclusive | Multiple licensees permitted | Lower royalty, broader use |
| Field of Use | Limited to specific application | Defined boundaries |
| Territorial | Geographic limitations | Clear boundaries |

### 7.2 Assignment Considerations

```
IP ASSIGNMENT CHECKLIST
=======================
□ All IP rights identified (patents, TMs, copyrights, trade secrets)
□ Chain of title verified
□ Recorded assignments current
□ Employee invention assignments in place
□ Contractor IP assignment provisions
□ Third-party licenses identified (in-bound and out-bound)
□ Encumbrances identified
□ Representations and warranties appropriate
□ Post-closing obligations clear
□ Record assignment with appropriate offices
```
