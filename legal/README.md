| **Conditions** | Include copyright notice, Include license copy, State significant changes, Include NOTICE file |
| **Limitations** | No trademark rights, No warranty, No liability assumed |

### Liability Framework

| Professional Domain | Required Oversight | Key Regulations |
|--------------------|-------------------|-----------------|
| Medical/Clinical | Licensed Physician (MD/DO) | FDA, HIPAA, State Medical Boards |
| Legal | Licensed Attorney (Bar Admission) | ABA Model Rules, State Bar Rules |
| Financial | Licensed Professional (CFP, CFA, CPA) | SEC, FINRA, State Securities |
| Scientific | Qualified Researcher (PhD) | IRB, IACUC, NIH Guidelines |
| Security | Certified Professional (CISSP, CISM) | GDPR, CCPA, Industry Standards |

**ALL FINAL DECISIONS AND LIABILITY REST WITH USER**
