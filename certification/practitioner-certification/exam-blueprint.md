## Question Types

### Multiple Choice

Standard multiple choice with 4 options, 1 correct answer.

```
Q: [Question stem]
A) [Option A]
B) [Option B]
C) [Option C]
D) [Option D]
```

### Scenario-Based

Situational questions requiring application of knowledge.

```
Scenario: [Description of situation]
Q: What is the most appropriate action?
A) [Option A]
B) [Option B]
C) [Option C]
D) [Option D]
```

### Multiple Response

Questions with multiple correct answers.

```
Q: [Question stem] (Select all that apply)
[ ] Option A
[ ] Option B
[ ] Option C
[ ] Option D
```

### Matching

Matching items from two lists.

```
Match each engine to its primary purpose:
1. Oracle Layer      A. Strategic analysis
2. CEREBRO-Lite      B. Fact verification
3. Decision Engine   C. Multi-framework analysis
4. Strategy Engine   D. Decision structuring
```

### Ordering

Putting steps in correct sequence.

```
Q: Order the following steps for engine stacking:
[ ] Verify with Oracle Layer
[ ] Identify problem
[ ] Select primary engine
[ ] Review and refine output
[ ] Apply secondary engine
```
