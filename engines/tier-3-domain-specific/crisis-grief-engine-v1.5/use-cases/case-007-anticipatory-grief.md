# Use Case 007: Anticipatory Grief Support Framework

**Engine:** Crisis & Grief Engine v1.5
**Domain:** Pre-Loss Grief
**Complexity:** High
**Risk Level:** HIGH (end-of-life context)

---

## Scenario Description

A hospice social worker needs educational content about anticipatory grief for family education. The engine provides psychoeducational frameworks for professional use, not direct patient/family counseling.

---

## Oracle Layer Self-Correction Protocol

```

## Input/Output Example

### Engine Response
```
ANTICIPATORY GRIEF EDUCATION FRAMEWORK
======================================

[PROFESSIONAL EDUCATIONAL RESOURCE]

SECTION 1: UNDERSTANDING ANTICIPATORY GRIEF
───────────────────────────────────────────

Definition:
Grief experienced before a death occurs, typically when
a loved one has a terminal diagnosis.

Key Characteristics:
├─ GRIEF BEFORE LOSS
│   ├─ Not "practice grief" — it's real grief
│   ├─ Mourning the loss of the future
│   ├─ Grieving changes already occurring
│   └─ May include guilt about grieving early
│
├─ DUAL REALITY
│   ├─ Person is still alive AND dying
│   ├─ Hope AND acceptance coexist
│   ├─ Planning for death AND living in present
│   └─ This tension is normal, not dysfunction
│
├─ ROLE CHANGES
│   ├─ Caregiver burden
│   ├─ Changing relationship dynamics
│   ├─ Anticipating identity shift
│   └─ Financial and practical concerns
│
└─ NOT PREDICTIVE
    ├─ Experiencing anticipatory grief doesn't "use up" grief
    ├─ Post-death grief will still occur
    ├─ May be complicated OR may facilitate adjustment
    └─ Individual variation is significant

COMMON EXPERIENCES:

Emotional:
├─ Sadness, depression
├─ Anxiety about the dying process
├─ Anger (at disease, God, healthcare, self)
├─ Guilt (for many reasons)
├─ Relief (often with more guilt)
├─ Ambivalence
└─ Numbness or emotional exhaustion

Physical:
├─ Sleep disturbance
├─ Appetite changes
├─ Fatigue
├─ Somatic symptoms
└─ Immune system impact

Cognitive:
├─ Preoccupation with dying person
├─ Difficulty concentrating
├─ Rehearsing death scenarios
├─ Planning (funeral, finances, etc.)
└─ Memory and decision-making challenges

Social:
├─ Isolation
├─ Changed relationships
├─ Communication difficulties
└─ Support system strain

SECTION 2: FAMILY EDUCATION PRINCIPLES
──────────────────────────────────────

Messages for Families:

1. NORMALIZE THE EXPERIENCE
├─ "What you're feeling is a normal response"
├─ "Many people in your situation feel this way"
├─ "There's no right or wrong way to do this"
└─ Validate without directing

2. PERMISSION TO GRIEVE
├─ "You can grieve while your loved one is still here"
├─ "Grieving now doesn't mean you're giving up"
├─ "You can still hope while also preparing"
└─ Reduce guilt about "pre-grieving"

3. SELF-CARE IS NOT SELFISH
├─ "You can't pour from an empty cup"
├─ "Your loved one would want you to care for yourself"
├─ "Taking breaks is necessary, not abandonment"
└─ Caregiver burnout prevention

4. EACH PERSON'S GRIEF IS UNIQUE
├─ "Family members may grieve differently"
├─ "Different relationships mean different grief"
├─ "No one is doing it wrong"
└─ Reduce family conflict over "correct" grieving

5. PROFESSIONAL SUPPORT EXISTS
├─ "You don't have to do this alone"
├─ "Support is available and encouraged"
├─ "Asking for help is strength"
└─ Connect to hospice psychosocial team

SECTION 3: OPPORTUNITIES DURING THIS TIME
─────────────────────────────────────────

Potential Positive Aspects:
(Not to minimize grief, but to illuminate possibilities)

├─ TIME FOR GOODBYE
│   ├─ Express love and gratitude
│   ├─ Resolve unfinished business
│   ├─ Create final memories
│   └─ Say what needs to be said
│
├─ HONORING WISHES
│   ├─ Ensure comfort care preferences known
│   ├─ Discuss funeral/memorial wishes
│   ├─ Document legacy (stories, letters)
│   └─ Fulfill bucket list items if possible
│
├─ FAMILY CONNECTION
│   ├─ Opportunity for reconciliation
│   ├─ Strengthening bonds
│   ├─ Shared caregiving
│   └─ Meaning-making together
│
└─ PREPARATION
    ├─ Practical arrangements
    ├─ Emotional preparation (though incomplete)
    ├─ Support system mobilization
    └─ Gradual adjustment begins

SECTION 4: RED FLAGS REQUIRING ADDITIONAL SUPPORT
─────────────────────────────────────────────────

When to Escalate to Clinical Team:

├─ Suicidal ideation in family member
├─ Severe depression (can't function)
├─ Substance abuse escalation
├─ Complete avoidance/denial impacting care
├─ Significant family conflict affecting patient
├─ Caregiver collapse/burnout
├─ Pre-existing mental health decompensation
└─ [MANDATORY: Hospice team notification]

[SAFETY_PRIORITY]
Any indication of harm to self or others
requires immediate clinical intervention.

SECTION 5: CULTURAL CONSIDERATIONS
──────────────────────────────────

Cultural Humility Required:
├─ Grief expression varies by culture
├─ Family decision-making norms differ
├─ Death and dying beliefs vary
├─ Rituals and observances differ
├─ Ask, don't assume
└─ [VERIFY_REQUIRED:cultural_consultation]

Examples (Not Comprehensive):
├─ Some cultures: Open emotion expression valued
├─ Some cultures: Stoic containment valued
├─ Some cultures: Community involvement expected
├─ Some cultures: Family-only appropriate
└─ Individual variation within any culture

└─ Hospice interdisciplinary team coordination

[VERIFY_REQUIRED:professional_use_only]
