# Criminal Defense Module (CDM-v1.0)

**Classification:** Legal Engine v2.2 Enhancement Module
**Purpose:** Criminal defense analysis and constitutional rights protection

---

## 1. Module Overview

The Criminal Defense Module provides frameworks for analyzing criminal charges, constitutional defenses, evidentiary issues, and sentencing considerations while maintaining strict ethical boundaries.

### Core Functions

| Function | Description |
|----------|-------------|
| Charge Analysis | Element breakdown and defense identification |
| Constitutional Review | Rights violation detection |
| Evidence Assessment | Admissibility and suppression analysis |
| Sentencing Analysis | Guidelines and mitigation |
| Plea Evaluation | Plea bargain analysis framework |

## 4. Evidence Assessment

### 4.1 Admissibility Framework

| Evidence Type | Key Issues | Defense Challenges |
|---------------|------------|-------------------|
| Eyewitness ID | Suggestiveness, reliability | Wade/reliability factors |
| Confessions | Voluntariness, Miranda | Suppression motion |
| Physical Evidence | Chain of custody, testing | Foundation, methodology |
| Expert Testimony | Daubert/Frye | Methodology challenge |
| Character Evidence | 404(b) other acts | Unfair prejudice |
| Hearsay | Exceptions, confrontation | Object/Crawford challenge |

### 4.2 Brady/Giglio Checklist

```
DISCOVERY OBLIGATIONS CHECKLIST
===============================
□ Exculpatory evidence requested and received
□ Impeachment material for government witnesses
□ Prior inconsistent statements of witnesses
□ Benefits/deals given to cooperating witnesses
□ Witness criminal histories
□ Evidence of witness bias or motive
□ Reports of experts not being called
□ Prior bad acts of defendant (if 404(b) notice required)

POTENTIAL BRADY VIOLATIONS IDENTIFIED:
[List any suspected withheld material]

REMEDY: [Motion to compel/Motion to dismiss/Mistrial]
```
