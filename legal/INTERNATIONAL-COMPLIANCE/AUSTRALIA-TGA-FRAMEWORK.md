## Australian Regulatory Framework

### Therapeutic Goods Act 1989

The TGA regulates therapeutic goods including medical devices under this legislation.

### Classification System

| Class | Risk Level | Pathway | Examples |
|-------|-----------|---------|----------|
| **Class I** | Lowest | Manufacturer Declaration | Non-invasive devices |
| **Class I (Measuring)** | Low | TGA conformity assessment | Blood pressure monitors |
| **Class IIa** | Low-moderate | TGA conformity assessment | Diagnostic equipment |
| **Class IIb** | Moderate-high | TGA conformity assessment | Infusion pumps |
| **Class III** | High | TGA conformity assessment | Implants |
| **AIMD** | Active Implantable | Full assessment | Pacemakers |

### Software as Medical Device

```
TGA SOFTWARE CLASSIFICATION
═══════════════════════════════════════════════════════════════

Based on intended purpose and risk:

Clinical Decision Support
├─► Life-threatening decisions → Class III
├─► Serious clinical decisions → Class IIb
├─► Moderate clinical impact → Class IIa
└─► General information → Class I or excluded

AI/ML-based Software
├─► Follows IMDRF SaMD framework
├─► Requires clinical evidence
├─► Post-market monitoring required
└─► Change management procedures

═══════════════════════════════════════════════════════════════
AION-BRAIN: NOT CLASSIFIED — Research framework only
```
