- Scan for judgment biases (anchoring, availability, representativeness)

- Assess position strength (HIGH/MEDIUM/LOW)

### Framework 4: PEARL (Causal Inference)
- Distinguish correlation from causation
- Map causal relationships (who/what causes what)
- Identify confounders (variables affecting both cause and effect)
- Apply counterfactual reasoning (what WOULD have happened if...)
- Rate causal confidence (STRONG/MODERATE/WEAK/NONE)

### Framework 5: TALEB (Anti-Fragility & Stress Testing)
- Classify as FRAGILE/ROBUST/ANTI-FRAGILE
- Run stress scenarios (10x spike, 90% collapse, dependency failure)
- Check for optionality (ability to benefit from surprises)
- Assess skin in the game (who bears consequences)
- Recommend via negativa (what to remove)

- Do NOT let one framework's findings influence another during analysis
- Complete all 5 analyses first, THEN synthesize

## Tips for Best Results

1. **Be specific** — Vague inputs produce vague outputs
2. **Provide context** — Background information improves accuracy
3. **State your goal** — What decision are you trying to make?
4. **Ask for uncertainty** — Request confidence levels explicitly
5. **Validate STRONG patterns** — Even 3+ framework agreement isn't certainty
6. **Re-run for changes** — Patterns decay; update analysis when conditions shift
